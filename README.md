## Overview

Sere Innovations is a conversion-focused marketing website designed for small and mid-sized poultry farmers. The experience emphasizes trust, simplicity, and affordability, guiding visitors from product awareness to enquiry with a clear, low-friction narrative.

The site is available in English and Hindi and includes a smooth-scrolling, mobile-friendly interface tailored for practical decision-making rather than flashy product hype.

## Highlights

- Bilingual experience with English and Hindi content switching
- Strong storytelling flow from farmer pain points to product solution
- Feature and benefits sections written for a non-technical audience
- Savings calculator to help estimate cost impact
- FAQ section to reduce objections before contact
- Enquiry form and WhatsApp call-to-action for fast lead capture
- Smooth, polished scrolling with Lenis
- Responsive design for mobile, tablet, and desktop

## Getting Started

### Prerequisites

- Node.js 18+
- npm 10+

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Production Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```
